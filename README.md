## Como Executar



### Pré-requisitos
- Flutter 3.32.6 • channel stable
- Dart 3.8.1

### Passos Rápidos

1. **Clone e entre na pasta**
2. **Instale as dependências**
   
   ```bash
   flutter pub get

3. **Gere o código MobX .g.dart**

   ```bash
   dart run build_runner build --delete-conflicting-outputs

4. **Execute**

   ```bash
   flutter run
